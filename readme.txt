a3_analyzer.py is main Python program for R1
r2_report.pdf is written for R2

To run the program:

  python3 a3_analyzer.py <pcap_file>

Examples:
  python3 a3_analyzer.py traceroute-frag.pcap
  python3 a3_analyzer.py win_trace1.pcap
  python3 a3_analyzer.py win_trace2.pcap
  python3 a3_analyzer.py group1-trace1.pcap
  python3 a3_analyzer.py group2-trace1.pcap
